Errors and status codes

All errors are returned in the following format:

{
  "status": "error",
  "message": {
    "not_found": "Breed not found."
  }
}
Status code of response corresponds to the error type.

API Endpoints

Get list of available breeds:

GET {{ site_url }}api/breeds

Returns 'Multiple breeds'

Add a new breed to the list:

POST {{ site_url }}api/breeds

Example request body:

{
  "breed": {
    "name": "cornish rex"
  }
}

Returns added 'Cat breed'

Get a list of images of the specific breed:

GET {{ site_url }}api/breeds/:breed

Returns 'Multiple images'

Get a random image of the specific breed:

GET {{ site_url }}api/breeds/:breed/random

Returns a 'Cat image'

Upload an image of the specific cat breed:

POST {{ site_url }}api/breeds/:breed

Use multipart/form-data to upload the image. Image size is limited to 500KB. You should name your file(s) with the same name `files` in request, for example: 

  curl -F files=@image1.jpg -F files=@image2.png localhost:8000/api/breeds/british-longhair

Returns added 'Cat image'
